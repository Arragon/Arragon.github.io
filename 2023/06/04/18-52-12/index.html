2023-06-04

生成时报错：extends includes/layout.pug block content #recent-posts.recent-posts include includ

  1. 在博客目录下打开git bash
  2. 输入命令：npm install –save hexo-renderer-jade hexo-generator-feed hexo-generator-sitemap hexo-browsersync hexo-generator-archive
  3. hexo clean（清除缓存）
  4. hexo g（生成静态文件）
  5. hexo s
  6. hexo d(部署到GitHub上)
